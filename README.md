# Rockbuster Stealth SQL Analysis

**Project:** CareerFoundry Data Immersion Program – Achievement 3  
**Objective:** Analyze Rockbuster Stealth’s movie rental data to support the launch of an online video rental service and inform data-driven business decisions.

---

## Key Questions
- Which movies contributed the most or least to revenue?  
- What was the average rental duration for all videos?  
- Which countries are Rockbuster customers based in?  
- Where are high lifetime value customers located?  
- Do sales vary across geographic regions?

---

## Data - Refer to Project Brief for links
- Rockbuster film inventory, customer, and payment datasets (~3MB)  
- Loaded into **PostgreSQL** relational database for analysis  

---

## Tools & Skills
- Database concepts: relational schema, keys, indexes, ETL, data dictionary  
- **SQL (PostgreSQL)** for querying and analysis

## Methodology
1. **Database Setup:** Installed PostgreSQL, loaded data into relational tables  
2. **Data Exploration:** Reviewed table structure, relationships, and key variables  
3. **SQL Analysis:**  
   - Basic queries (CRUD operations)  
   - Filtering and grouping (WHERE, HAVING, CASE statements)  
   - Joining multiple tables and subqueries  
   - Common Table Expressions (CTEs) for complex queries  
4. **Data Cleaning & Summarization:** Identified dirty data, created summary statistics, and used views  
5. **Visualization & Reporting:** Generated charts and dashboards to present results to management  

---

## Insights
- Identification of top spending countries and customer distribution
- Top-performing genres movies and ratings identified  
- Inventory Analysis
- Statistics on rental durations across top 10 countries  
- Sales performance differs between regions, highlighting growth opportunities  

---

## Recommendations
- Focus on key top 5-10 countries before expanding  
- Small need to invest in inventory at the moment
- Focus on top clients for success
